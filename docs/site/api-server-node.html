API

Server Node API reference

Server Node executes guild state. Includes channels, messages, emotes, attachments, voice flow, extension runtime, and moderation actions.

Guilds, channels, messages

MethodPathPurpose
GET/v1/guildsList memberships
POST/v1/guildsCreate guild
GET/v1/guilds/:guildId/channelsList channels (hidden-channel aware)
POST/v1/guilds/:guildId/channels/reorderReorder channels/categories
POST/v1/channels/:channelId/messagesSend message
DELETE/v1/channels/:channelId/messages/:messageIdDelete message

Attachments, embeds, emotes

MethodPathPurpose
POST/v1/attachments/uploadTier-aware upload (boost perks supported)
GET/v1/attachments/:idFetch attachment object
GET/v1/guilds/:guildId/emotesList custom server emotes
POST/v1/guilds/:guildId/emotesCreate custom emote
DELETE/v1/guilds/:guildId/emotes/:emoteIdDelete emote

Voice and moderation

MethodPathPurpose
POST/v1/channels/:channelId/voice/joinJoin voice channel
POST/v1/channels/:channelId/voice/leaveLeave voice channel
POST/v1/guilds/:guildId/members/:memberId/kickKick member
POST/v1/guilds/:guildId/members/:memberId/banBan member
DELETE/v1/guilds/:guildId/bans/:memberIdUnban member

Node extension runtime

  • GET /v1/extensions and GET /v1/extensions/commands
  • GET/PUT /v1/extensions/:extensionId/config
  • POST /v1/extensions/activate and POST /v1/extensions/:extensionId/deactivate
  • POST /v1/extensions/commands/:commandName/execute

Source routes live in backend/packages/server-node/src/routes.
