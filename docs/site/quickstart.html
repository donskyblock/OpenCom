Quickstart

Minimal setup path for local development on Linux, macOS, and Windows.

Prerequisites

  • Node.js 20+
  • npm 10+
  • Docker + Docker Compose (recommended)

Clone and setup

git clone <repo-url>
cd OpenCom
cp backend/.env.example backend/.env
./scripts/dev/setup.sh all

Run services

./scripts/dev/start.sh all

Windows commands

scripts\setup.bat all
scripts\start.bat all

Database migrations

./scripts/dev/setup-database.sh
# or full provision:
./scripts/dev/setup-database.sh --init-env --with-docker
