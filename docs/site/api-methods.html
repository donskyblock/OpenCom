API

Complete Method Index

Endpoint inventory for Core API, Server Node API, and Discord-compat routes. Use this as the definitive route map when implementing clients, tools, extensions, or bots.

Conventions

  • Unless noted, endpoints require Authorization: Bearer <accessToken>.
  • Core routes are hosted on Core API. Guild-scoped runtime routes are hosted on Server Node API.
  • Route behavior and payload validation come from files in backend/packages/core/src/routes and backend/packages/server-node/src/routes.
